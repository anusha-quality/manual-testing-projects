# Login Test Scenarios

## Positive Scenarios

1. Verify login with valid username and password.
2. Verify user is redirected to the dashboard after login.
3. Verify "Remember Me" functionality.

## Negative Scenarios

4. Verify error message for invalid password.
5. Verify error message for invalid username.
6. Verify login with blank username.
7. Verify login with blank password.
8. Verify login with both fields empty.

## Security Scenarios

9. Verify SQL injection protection.
10. Verify XSS attack prevention.
11. Verify account lock after multiple failed attempts.
