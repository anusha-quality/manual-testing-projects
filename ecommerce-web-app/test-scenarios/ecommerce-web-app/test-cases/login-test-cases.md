
# Login Test Cases

| TC ID  | Test Case        | Steps                                     | Expected Result                   |
| ------ | ---------------- | ----------------------------------------- | --------------------------------- |
| TC_001 | Valid Login      | Enter valid credentials and click Login   | User should navigate to Dashboard |
| TC_002 | Invalid Password | Enter correct username and wrong password | Error message should appear       |
| TC_003 | Blank Username   | Leave username empty                      | Validation message displayed      |
| TC_004 | Blank Password   | Leave password empty                      | Validation message displayed      |
| TC_005 | Remember Me      | Select Remember Me and login              | User session should persist       |
