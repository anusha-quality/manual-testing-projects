
# Sample Bug Reports

## BUG_001

**Title:** Login button remains enabled after multiple clicks.

**Severity:** Medium

**Priority:** High

**Steps to Reproduce:**

1. Open Login Page.
2. Enter valid credentials.
3. Click Login button repeatedly.

**Expected Result:**
Login button should disable after first click.

**Actual Result:**
Multiple requests are submitted.
